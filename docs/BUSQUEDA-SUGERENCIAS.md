# Sugerencias de búsqueda

La barra de búsqueda muestra sugerencias mientras el usuario escribe:

- **Productos**: Coincidencias por nombre (desde Medusa).
- **Sugerencias**: Palabras clave gestionadas desde Medusa (Product Tags).

## Controlar sugerencias desde Medusa

Las sugerencias de palabras clave se toman de los **Product Tags** de Medusa.

1. Entra al **Medusa Admin** (por ejemplo `http://localhost:9000/app`).
2. Ve a **Settings** → **Product Tags**.
3. Crea tags con las palabras que quieres que aparezcan como sugerencias:  
   Ejemplos: `grinder`, `pipas`, `bong`, `papel`, `conos`, `encendedor`, etc.
4. Asigna esos tags a los productos correspondientes.

Los tags `combo` y `featured` no se muestran como sugerencias (se excluyen internamente).
